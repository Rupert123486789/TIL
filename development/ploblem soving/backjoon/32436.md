# Enigma of the Jewelry Case

```python
import sys
input = sys.stdin.readline

n = int(input())
box = [list(map(int, input().split())) for _ in range(n)]

if box[0][0] < box[1][0]:
    if box[0][0] < box[0][1]:
        print(0)
    else:
        print(1)
else:
    if box[0][0] > box[0][1]:
        print(2)
    else:
        print(3)
```
