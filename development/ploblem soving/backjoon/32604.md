# Jumbled Scoreboards

```python
N = int(input())

ans_1 = 0
ans_2 = 0
flag = True
for _ in range(N):
    A, B = map(int, input().split())
    if ans_1 <= A and ans_2 <= B:
        ans_1 = A
        ans_2 = B
    else:
        flag = False
        break

if flag == True:
    print('yes')
else:
    print('no')
```
